# Viral Phylogeny

This repository contains code and dataset documentation for viral phylogenetic analysis and phylogenetic deep learning experiments.

## Folders

- [COVID_dataset](./COVID_dataset/README.md): documentation for the COVID dataset
- [PhyloBench](./PhyloBench/README.md): documentation for the PhyloBench dataset
- [Sample_data](./Sample_data/README.md): example data used for testing

## Scripts

- `compare.py`: compare phylogenetic inference results
- `remove_duplicates.py`: remove duplicated sequences from FASTA files
